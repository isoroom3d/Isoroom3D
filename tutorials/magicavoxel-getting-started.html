Getting started with MagicaVoxel

MagicaVoxel tutorial · Part 1 · Beginner

MagicaVoxel is a free, lightweight voxel editor that's perfect for building colorful, chunky isometric scenes. In this first tutorial, we'll install MagicaVoxel, get familiar with its interface, and place our first voxels.

1. Download MagicaVoxel

  1. Download MagicaVoxel from its official page (available for Windows and macOS).
  2. Unzip the downloaded folder — MagicaVoxel doesn't need a traditional installer, just run the executable inside.
  3. Launch the app. You'll see an empty voxel grid in the center of the screen.
Tip: MagicaVoxel is donation-based but completely free to use — you don't need to pay anything to access all features.

2. Get familiar with the interface

  • Viewport (center) — shows your voxel model. The default view is already close to isometric!
  • Tool palette (top left) — tools for adding, removing and painting voxels (Voxel, Face, Paint, etc.).
  • Color palette (right side) — pick the color/material applied by your current tool.
  • World / Model panel (bottom) — manage canvas size and multiple models within a scene.

To navigate: hold the right mouse button and drag to orbit, scroll to zoom, and hold middle mouse button to pan.

3. Set your canvas size

Before placing voxels, define how big your scene can be:

  1. In the bottom panel, find the Editor tab and the canvas size fields (e.g. 32 x 32 x 32).
  2. For a small room, a size like 24 x 24 x 24 is a good starting point — large enough for walls and furniture, small enough to stay manageable.

4. Place your first voxels

  1. Select the Voxel tool (usually labeled "V") from the tool palette.
  2. Pick a color from the palette on the right.
  3. Left-click on the grid to add a voxel. Left-click on an existing voxel's face to add another voxel attached to it.
  4. Right-click (with the Voxel tool) to remove a voxel.

Try building a simple floor: select a base color, then click across the grid to fill in a flat square — this will become the foundation of your room.

MagicaVoxel's default camera is already close to an isometric angle. You can fine-tune it using the rotation controls above the viewport, or press numeric keys to snap to preset views.

Next steps

With the basics down, you're ready to start building actual structures. In the next tutorial we'll combine voxels into walls, a floor, and simple furniture shapes.
