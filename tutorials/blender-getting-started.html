Getting started with Blender

Blender tutorial · Part 1 · Beginner

Blender is a free, open-source 3D creation suite available for Windows, macOS and Linux. In this first tutorial, we'll install Blender, get familiar with the interface, and set up a camera with an isometric (orthographic) view — the foundation for every isometric room.

1. Install Blender

  1. Go to the official Blender website and download the latest version for your operating system.
  2. Run the installer and follow the default steps.
  3. Launch Blender. You should see the default scene with a cube, a light and a camera.
Tip: Blender is completely free, with no trial period or feature restrictions. Always download it from the official source.

2. Get familiar with the interface

Blender's main window is divided into a few key areas:

  • 3D Viewport — the large central area where you see and interact with your scene.
  • Outliner — top right, lists every object in your scene (camera, lights, meshes).
  • Properties panel — bottom right, lets you edit settings for the selected object, render, and scene.
  • Timeline — bottom, used for animation (not needed for a static isometric room).

To move around the viewport: hold the middle mouse button and drag to orbit, scroll to zoom, and hold Shift + middle mouse button to pan.

3. Set up an isometric camera

A true isometric view uses an orthographic camera (no perspective distortion) angled so that all three axes appear equally foreshortened. Here's how to set it up:

  1. Select the default camera in the Outliner.
  2. In the Properties panel, click the camera icon (Object Data Properties).
  3. Under Lens, change the type from Perspective to Orthographic.
  4. Rotate the camera to a classic isometric angle: set its rotation to X = 54.7°, Y = 0°, Z = 45° in the Object Properties tab.
  5. Position the camera away from the origin (e.g. X = 10, Y = -10, Z = 10) so it looks toward your scene.
  6. Press Numpad 0 to preview the view through the camera at any time.
A handy shortcut: with the camera selected, press Ctrl + Numpad 0 to snap your current viewport view to become the camera's view — useful once you've manually framed a nice angle.

4. Adjust the orthographic scale

The Orthographic Scale property (next to the Lens type) controls how much of the scene is visible — think of it as zoom. Increase it to fit more of your room in frame, decrease it to zoom in on details.
